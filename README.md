# VCAL Semantic Cache (Former VCAL Server) Releases

This repository hosts public release information for **VCAL Semantic Cache**.

VCAL Semantic Cache is a commercial, closed-source product from VCAL Labs.  
This repository does **not** contain the VCAL Semantic Cache source code.

## What this repository is for

This repository may be used for:

- publishing VCAL Semantic Cache release notes
- linking to downloadable binaries or Docker images
- providing checksums and release metadata
- tracking public version history

## What this repository is not for

This repository is not used for:

- hosting VCAL Semantic Cache source code
- accepting pull requests
- accepting feature requests or support issues
- publishing internal build scripts or private implementation details

## Downloads

VCAL Semantic Cache releases are distributed through official VCAL channels.

Recommended starting points:

- Documentation: https://server.docs.vcal-project.com/
- Product page: https://vcal-project.com/vcal-server/

Docker images and binaries may be linked from GitHub Releases when available.

## OpenAPI Specification

This repository may include a static `openapi.yml` file for VCAL Semantic Cache releases.

The standard free testing build of VCAL Semantic Cache does not expose the built-in OpenAPI/docs UI at runtime.

Customers who need runtime OpenAPI/docs UI access can request a custom build through VCAL Semantic Cache support.
